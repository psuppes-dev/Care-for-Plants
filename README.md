# Care-for-Plants
Webanwendung zum Pflanzenmanagement


Projektübersicht

Care for Plants unterstützt die strukturierte Verwaltung von Pflanzen, Standorten und Pflegeaufgaben. Nutzer können Pflanzen auf
einer Wunschliste erfassen, geeignete Standorte bestimmen, Pflanzen einem Standort zuordnen sowie automatisch generierte Pflegeaufgaben
verwalten. Darüber hinaus werden auf Basis vorhandener Standortbedingungen und des Pflanzenbestands regelbasierte Vorschläge für
geeignete, noch nicht vorhandene Pflanzen erstellt.


Zentrale Funktionen

•	Verwaltung einer Wunschliste für Pflanzen
•	Erfassung und Verwaltung von Standorten mit individuellen Eigenschaften
•	Regelbasierte Eignungsprüfung zwischen Pflanzen und Standorten
•	Verwaltung des Pflanzenbestands pro Standort und als Gesamtliste
•	Automatisch generierter Pflegekalender mit wiederkehrenden Aufgaben
•	Anzeige geeigneter, noch nicht vorhandener Pflanzen auf Basis des Bestands


Zugriff auf das System

Das Softwaresystem Care for Plants ist als Webanwendung umgesetzt und öffentlich erreichbar. Die Anwendung ist so konzipiert, dass sie
ohne lokale Installation oder zusätzliche Software direkt über einen gängigen Webbrowser genutzt werden kann.

Technische Voraussetzungen für den Zugriff:
•	Aktueller Webbrowser (z. B. Chrome, Firefox, Edge)
•	Internetverbindung
•	Keine zusätzliche Installation erforderlich
